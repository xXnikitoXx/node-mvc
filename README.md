## Installation
```
npm install @tech-lab/node-mvc
```

## Features
* **Database support**
    * **MongoDb**
    * More coming soon...
* **Model validation**
* **Dependency injection for services**
* **Render engine**
    * **Layouts**
    * **Templates**
    * **Variable interpolation**
    * **Conditional statements**
    * **Switch statement**
    * **For loop**
    * **HTTP requests**
